# Product

## Register

brand

## Users

普通大众，不限年龄。用户在主页/介绍页首次接触时以探索者身份进入（品牌接触，感受文化氛围）；进入游戏流程（工坊、排练、演出、回放）后切换为操作者身份，核心诉求是「能玩起来、玩得顺畅」。首要任务：降低传统文化的探索门槛，让从未接触过皮影戏的人也能在 10 分钟内完成一次完整的制影-演出体验。

## Product Purpose

「影幕西游」是一个以中国非遗皮影戏为主题的网页互动游戏，核心叙事围绕西游记「三打白骨精」展开。玩家依次经历：选择角色（孙悟空/唐僧/白骨精等）→ 皮影工坊（制皮、雕刻、上色、装关节）→ 幕后排练 → 正式演出（摄像头或键鼠操控影偶）→ 幕前回放（观看自己刚刚演出的光影）。

成功标准：玩家完成全流程后，感受到「我亲手做了一尊皮影，演了一出戏」的真实成就感，并对皮影戏这项非遗文化产生兴趣和尊重。

## Brand Personality

戏剧性 · 仪式感 · 沉浸入幕

- **戏剧性**：每一个界面都是一个「幕布场景」，像真实的皮影演出——有亮幕、有剧情、有定格。
- **仪式感**：制作影偶不是「填表单」，是一道道有名字的工序；开演不是「点按钮」，是「开锣」的时刻。
- **沉浸入幕**：用户应该感觉自己站在幕布后面，操纵灯影，而不是在用一个现代 App。

品牌色基于实际代码确认：深炭黑（#120B08）底色 + 琥珀金（#D99A2B）主调 + 米白（#F4E5C0）字色 + 朱砂红（#7A2E18）强调。这套色彩已由代码固定，是品牌的视觉锚点，后续设计以此为准。

## Anti-references

- **SaaS / 科技风**：蓝白渐变、大量阴影卡片、像素级圆角、Inter 字体全家——这些信号让人以为是一个效率工具，完全错位。
- **旅游宣传 / 政府网站**：大红大金、「非遗」二字在 banner 里撑满全屏、喜庆祥云——把文化变成了宣传物料，失去了游戏的私人感和仪式感。
- **通用游戏 UI（HUD 风）**：像素风、赛博发光、大量粒子和 VFX——脱离了皮影的「纸-光-影」质感，变成了电子游戏的视觉语言。
- **纸张/仿古米色模板**：白底 + 棕色边框 + 毛笔字图标——表面上「国风」，实际是最泛滥的 AI 生成美学，与项目现有的深色主题完全矛盾。

## Design Principles

1. **幕布先于 UI**：界面元素是舞台道具，不是 App 组件。颜色、版式、动效都应该首先问「在幕布上这合理吗？」
2. **流程即剧情**：每一步（制皮 → 雕刻 → 上色 → 装关节 → 演出）都有叙事重量。步骤转场是「幕间换场」，不是「页面跳转」。
3. **操作即表演**：玩家的交互动作（移动影偶、完成关节）应给予戏曲式反馈（定格感、亮相感），而不是常规的「进度条 + 勾选」。
4. **文化可信度优先于装饰**：引用的文字（工序名、戏曲术语）来自非遗资料，不随意创造。装饰元素要能说清楚它对应的真实皮影工艺细节。
5. **轻盈胜过厚重**：皮影的魅力在于「光穿透薄皮的透亮」，设计也应该有这种透明感——用透明度和光晕，而不是实心色块的堆叠。

## Accessibility & Inclusion

WCAG 2.1 AA：
- 正文文字对比度 ≥ 4.5:1（当前 #F4E5C0 on #120B08 满足，需验证 muted 状态）
- 大字 / 粗体 ≥ 3:1
- 所有动画提供 `prefers-reduced-motion` 替代方案（crossfade 或瞬间切换）
- 交互元素聚焦状态可见（焦点环不被完全隐藏）
- 关键游戏操作（演出控制）同时支持键盘路径
