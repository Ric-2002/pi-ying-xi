# 皮影戏

一个以《西游记・三打白骨精》为主题的 React 皮影互动网页游戏。玩家可以选择角色，完成制皮、雕刻、上色、装关节等工序，再进入排练和幕前表演流程。

## 功能

- 角色选择：孙悟空、唐僧、白骨精、猪八戒、沙僧
- 工坊流程：按皮影制作步骤完成角色准备
- 排练与演出：用键鼠或摄像头权限驱动动作练习
- 回放页面：查看一次表演的完成度
- 资料来源：保留非遗资料入口，方便追溯内容依据

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 数据迁移

项目目前处于 demo 阶段。重大架构升级(如 v0 → v2:从 `colors`/`jointQuality` 迁移到 `PuppetAsset`)采用**清空 localStorage** 的方式不向下兼容。如果之前玩过旧版本,首次打开新版会自动从空状态开始(zustand persist `version: 2` + `migrate` 钩子)。
