## 📝 报名流程

### 完整流程
```
用 TRAE Work 生成创意产物 HTML
        ↓
在论坛报名专区发「创意提案」帖
        ↓
官方每个工作日审核（只看合规，不评好坏）
        ↓
审核通过（官网 / 短信 / 社区公示三路通知）
        ↓
登录大赛官网确认领取速通奖励 ← ⚠️ 必须手动领取！
        ↓
进入初赛，制作可体验 Demo
```

### 报名帖格式要求

**【标签】** 按赛道选一个：`生活娱乐` / `学习工作` / `社会服务` / `硬件交互`（附加可加 `社会公益`）

**【标题】** `报名赛道 + 创意名称`
- 示例：`【学习工作赛道】AI 错题本——拍照自动整理孩子的错题`

**【正文】** 不少于 100 字，必须包含以下 4 个部分：

1. **创意名称 + 创意介绍**
   - 想解决什么问题（1-2句痛点描述）
   - 为什么会想到做这个（动机/灵感来源）
   - 大概是什么产品（App/小程序/网站/系统/硬件等）

2. **目标用户及痛点**
   - 面向哪些用户
   - 在什么场景下使用
   - 当前痛点是什么

3. **价值与意义**
   - 商业价值、效率提升、社会价值中选 1-2 个角度

4. **必须附带：** 上传 TRAE Work 生成的创意产物 HTML 文件

## 🔧 如何生成创意产物 HTML（零代码）

无需编程基础，只需与 TRAE Work 对话：

1. 打开 [TRAE Work](https://www.trae.cn/TRAE)，选择 **Auto 模式**
2. 用一句话描述创意，附上报名内容要求（创意介绍、目标用户、价值意义）
3. 等待 3-5 分钟，TRAE Work 自动生成完整创意提案 HTML
4. 将 HTML 文件上传到报名帖中

## ❓ 常见问题 FAQ

**Q：可以同时报名多个赛道吗？**
A：可以，也可以一个赛道报名多个创意。但每个创意单独发一个帖，报名通过奖励只发放一次。

**Q：报名审核没通过，可以修改旧帖重新审吗？**
A：不建议。官方只审核前一天新发的帖，建议直接重新发一篇。

**Q：收到报名通过通知但没收到奖励？**
A：需要去官网手动登录确认领取。另见「奖励领取失败」两种情况。

**Q：TRAE 中国版和社区账号为什么要用同一手机号？**
A：社区账号与 TRAE UID 一一对应，账号不匹配会导致资格校验失败或奖励无法发放。
