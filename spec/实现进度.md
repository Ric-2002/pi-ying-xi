### Hooks

| Hook | 说明 |
|------|------|
| `usePuppetControls` | 左右手坐标 → `PuppetPoseData` 数学映射（含 clamp、nudge、reset）|
| `useScriptTimeline` | `requestAnimationFrame` 驱动剧本时间轴，计算进度、当前 cue、姿态距离评分 |

| `useCameraPermission` | `getUserMedia` 封装，处理授权/拒绝/不支持三种状态，组件卸载自动释放流 |
| `useTheme` | 主题切换（基础实现）|

### 工坊四道工序（全部实现 ✅）

| 工序 | 实现内容 |
|------|---------|
| **制皮** | 画布上随机分布12个红点，依次点击模拟绷钉，剩余计数实时更新，全部点完自动进入下一步 |
| **雕刻** | SVG 虚线路径可点击（6处），点击后笔触固化（变实线+填色），全部完成才算通过 |
| **上色** | 10色传统矿物色盘 + 点击皮影色块区域（头/脸/身体/腰带/道具）填色，颜色实时写入 store |
| **装关节** | 3个关节滑块（左肩/右肩/颈部）调整初始角度，实时驱动 `PuppetFigure` SVG 旋转，点击「确认关节」写入 store 并完成 |

### 演出核心链路（`StagePage`）

- 「开演/暂停」控制时间轴启停
- 每帧记录 `PerformanceFrameData`（时间戳 + 姿态 + 当前 cueId）
- 实时 `scorePose()` 与目标姿态做距离计算并展示
- **唱词字幕**：跟随当前 cue 的 `lyric` 字段在皮影上方叠加显示
- **烟雾特效**：当前 cue `smoke: true` 时触发烟雾层动画
- **幕前幕后切换**：按钮切换 `isFrontView`，幕前隐藏控制板，幕后显示控制板
- 演出结束后自动调用 `savePerformance` 并跳转到 `/replay/:id`

### 回放核心链路（`ReplayPage`）

- **帧插值动画**：`useReplayAnimation` hook 使用 `requestAnimationFrame` 按时间戳对帧数组做线性插值，完整重播整段演出
- **唱词同步**：随时间轴推进，底部显示当前 cue 的唱词和场景名
- **烟雾特效**：与演出页相同的烟雾层，根据 `activeCue.smoke` 触发
- **得分评级**：5档评语（天人之影/匠心之影/匠人之影/学徒之影/初心之影）
- 「再看一遍」和「再做一个角色」按钮

### 剧本数据

- `gameData.ts`：5个角色、4道工序、默认姿态、**9条剧本 cue**（总时长 58 秒）、3条资料来源
- 9条 cue 完整覆盖：白骨精初现 → 悟空识妖 → 一棒挥出 → 二次化身 → 悟空再识 → 唐僧念咒 → 三化老翁 → 三打定胜负 → 妖影散去
- 每条 cue 包含：`lyric`（唱词）、`smoke`（是否触发烟雾）、`targetPose`、`tolerance`
- `SHOW_DURATION_MS = 58000` 常量导出供 hook 使用

### 数据层

- `types/game.ts`：TypeScript 类型全量定义，`ScriptActionData` 新增 `lyric`、`smoke` 字段

## 三、未完成 / 待优化 ⚠️

### 1. 排练页摄像头无手势识别

**现状**：摄像头打开后只是 `<video>` 原始预览，没有图像分析，不能驱动皮影。

**单文件版已实现**：基于肤色阈值（R > G > B && R > 95）+ 逐帧亮度差的运动检测，识别画面左右两半的手部重心坐标，映射到 `livePose.leftArm` 和 `livePose.rightArm`。

**React版待实现**（P4，可选）：将识别逻辑封装为 `useCameraGesture` hook，输出 `leftHand` / `rightHand` 坐标，接入现有 `usePuppetControls` 的相同接口。

### 2. 皮影外观与角色无关（P3，视觉品质）

**现状**：`PuppetFigure` 只有一套通用 SVG 形状，5个角色只是颜色不同，外形无差异（无头冠、无武器外形差异）。

**单文件版已实现**：`buildPuppetSVG(roleId)` 按角色 id 生成不同头冠、脸谱眉形、武器 SVG（金箍棒/禅杖/九齿钉耙/月牙铲/袖剑）。

**待实现**：为 `PuppetFigure` 增加角色差异化 SVG 路径，至少区分孙悟空（金箍棒+花冠）和白骨精（飘袖+骨冠）。
